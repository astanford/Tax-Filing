# Private Data Guide

This document describes the directories and files that contain personal financial data and are **not** included in this repository. You'll create these locally when using the skills.

## `my-tax-docs/`

Place your scanned or PDF tax documents here. The `/tax-prep` skill reads from this directory.

**Expected contents:**

| Document Type | Example Filename |
|--------------|-----------------|
| W-2 (wages) | `w2-acme-corp-2025.pdf` |
| 1099-INT (interest) | `1099-int-first-national-2025.pdf` |
| 1099-DIV (dividends) | `1099-div-vanguard-2025.pdf` |
| 1099-B (brokerage) | `consolidated-1099-brokerage-2025.pdf` |
| 1099-K (payment platform) | `1099-k-platform-2025.pdf` |
| 1098 (mortgage) | `1098-mortgage-2025.pdf` |
| 1098-E (student loan) | `1098-e-student-loan-2025.pdf` |
| 1095-C (health insurance) | `1095-c-employer-2025.pdf` |
| Charitable receipts | `donation-receipt-charity-2025.pdf` |
| Rental property records | `llcb-annual-statement-2025.pdf` |
| Prior-year returns (full, with PII) | `prior-years/2024-tax-return.pdf`, `prior-years/2023-tax-return.pdf` |

**Tips:**
- Redact SSNs before scanning if you prefer — the skills never need them
- Organize by institution in subfolders if you have many documents
- The skill handles most PDF formats, but clear scans work best

## `analysis/`

Generated by the skills during your workflow. You don't create this manually.

**Generated files:**

| File | Created By | Contents |
|------|-----------|----------|
| `tax-doc-summary.csv` | `/tax-prep` | Every extracted box/line value from your documents |
| `cheatsheet-{form}.md` | `/tax-cheatsheet` | Line-by-line guidance for each form |
| `analysis-cheatsheet-*.md` | `/tax-cheatsheet` | Detailed cheat sheets with your values |
| `analysis-schedule-c-breakdown-*.csv` | `/tax-cheatsheet` | Business expense categorization |
| `prior-year-carryovers-<year>.json` | `/tax-prep` (or your local Hermes Agent) | Prior-year carryovers — amounts only, PII-scanned by `validate_prior_year.py` (see [PRIOR-YEAR-DATA.md](PRIOR-YEAR-DATA.md)) |

The CSV is the single source of truth. See [templates/tax-doc-summary-template.csv](../.claude/skills/tax-prep/templates/tax-doc-summary-template.csv) for the format with sample data.

## `reference/Raw/`

Original IRS publications and state tax materials. Not included because they are large (~40MB) and freely available.

**To download:**
- Form 1040 instructions: [irs.gov/instructions/i1040](https://www.irs.gov/instructions/i1040)
- Publication 17: [irs.gov/publications/p17](https://www.irs.gov/publications/p17)
- Publication 550: [irs.gov/publications/p550](https://www.irs.gov/publications/p550)
- Publication 936: [irs.gov/publications/p936](https://www.irs.gov/publications/p936)
- Schedule C instructions: [irs.gov/instructions/i1040sc](https://www.irs.gov/instructions/i1040sc)
- Georgia IT-511 booklet and Form 500: [dor.georgia.gov](https://dor.georgia.gov) (2025 copies committed at reference/Raw/)

These raw sources are used to create the curated reference files in `reference/curated/`. See [HOW-TO-CURATE.md](../reference/HOW-TO-CURATE.md) for the curation process.

## Security Notes

- The `.gitignore` blocks `my-tax-docs/`, `analysis/`, and `reference/Raw/` from being committed
- Skills are designed to never store SSNs, account numbers, or routing numbers
- Generated forms leave SSN, bank, and signature fields blank — you fill those in by hand
- If you fork this repo, verify your `.gitignore` is intact before pushing
